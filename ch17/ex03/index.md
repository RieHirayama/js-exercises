## 問題 17.3 🖋️

npm に同梱されている npx を利用することにはどのような利点があるのか説明しなさい。

## 回答

npx は npm に同梱されているコマンド実行ツールであり、ローカルまたは一時的にインストールしたパッケージを直接実行できる。  
npx を利用する利点は以下の通りである。

- グローバルインストールが不要  
npx を利用すれば、グローバルインストールせずにコマンドを実行できる。  
これにより、環境を汚さない、バージョン衝突を防げる、不要なパッケージが残らないという利点がある。  

- ローカルインストール済みパッケージを簡単に実行できる  
プロジェクト内にインストールされた./node_modules/.binに存在するパッケージを PATH を意識せずに実行できる。  
これにより、scripts に書かなくても一時的に実行可能である。  

- 一時的にパッケージをダウンロードして実行できる  
npx は、ローカルに存在しないパッケージであっても、その場でダウンロードして実行できる。  
一度きりの利用や検証に非常に便利。  

- バージョンを指定して実行できる  
特定のバージョンを指定して実行できる。  
これにより、古いプロジェクトの検証ができるなど便利。
