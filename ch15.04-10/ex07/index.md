## 問題
.png 画像や .jpeg 画像と比べた際の SVG のメリット・デメリットを調査しなさい。

## 回答
### SVG のメリット
- 拡大・縮小しても劣化しない（ベクター形式）
- ファイルサイズが小さくなりやすい
- CSS / JavaScript で色や形を自由に変えられる
- 画像でも検索対象になる(SVG の中のテキスト要素はそのまま “文字” なので）

### SVG のデメリット
- 写真などの多色・複雑な画像には向かない
- 複雑な SVG は表示コストが高い
- セキュリティ観点で注意が必要(XML + JavaScript を含むことができるため、XSS 攻撃が可能)
